## 4. DATA STORAGE

### 4.1 Where Data is Stored
- **Google Sheets**: All attendance and user data is stored in Google Sheets owned by your hostel
- **Local Storage**: Temporary drafts are stored in your browser
- **Google Cloud**: Backend processing through Google Apps Script

### 4.2 Data Retention
- Active data: Stored as long as your account exists
- Archived data: Moved to archive sheets (accessible by admin)
- Deleted data: Permanently removed when purged by admin

### 4.3 Data Security Measures
- Password hashing using SHA-256 encryption
- One-Time Password (OTP) for verification
- Session-based authentication
- Each hostel's data is isolated in separate sheets
- HTTPS encryption for all data transfer
